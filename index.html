who i am//

I'm a 26yo self-taught webdev from Cornwall/England, but now reside in Stockholm/Sweden with my incredible partner, Rafael, and our tiny fluff ball of a dog, Leeloo.

I've always enjoyed web development, however it wasn't until April 2016 that I really started my quest to become a full stack web dev. I started with The Odin Project, an open source curriculum. After flirting around with that for a few months, I discovered Free Code Camp. If you want to learn to code this is an ideal place to start. It’s an open source curriculum that is vast, active and intensive with a large supportive community and non profit projects. They push you to your limits, none of this easy hand holding rubbish. It’s all sweat, blood and tears.

As of now I am focused on front-end development and, with help from the above sources, I have taught myself HTML - CSS(3) - jQuery - Bootstrap with an understanding of JavaScript, to create simple, clean and responsive webpages. I'll admit I've still got a lot to learn, but haven't we all?

Chosen OS: Ubuntu, but I can work with any.

When I'm not pushing code you can find me playing computer games, mostly World of Warcraft (*sigh* I know, I have a problem), socializing with friends, drinking tea, collecting mugs for said tea, catching up on my reading material (currently on The Pragmatic Programmer) and out walking my pupper.

tldr: I’m self taught in HTML, CSS(3), Bootstrap, JavaScript, jQuery, Git. I use Ubuntu. I’m a computer gamer, dog walker, socializer, pub goer, book reader, and tea enthusiast.
