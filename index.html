who i am//

I'm a 27yo self-taught webdev from Cornwall/England, but now reside here in Stockholm/Sweden with my incredible partner, Raffe, and our tiny fluff ball of a dog, Leeloo.

I've always enjoyed web development, however it wasn't until late 2016 that I discovered Free Code Camp. If you want to learn to code then this is an ideal place to start. It’s an open source curriculum that is intensive with a large supportive community. As of now, I am mainly focused on front-end development and, with help from the above, I have taught myself HTML, CSS, SCSS, Bootstrap, JavaScript and jQuery. I’m currently experimenting with the likes of Pug, React and Redux, but I'll admit I've still got a lot to learn, but that's what's so exciting!

When I'm not pushing code you can find me playing Rugby with the Stockholm Berserkers, occasionally playing computer games, socializing with friends, drinking tea, catching up on my reading material and out walking/jogging with my dog.
