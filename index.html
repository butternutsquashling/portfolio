who i am//

I'm a 26yo self-taught webdev from Cornwall/England, but now reside here in Stockholm/Sweden with my incredible partner, Rafael, and our tiny fluff ball dog, Leeloo.


I've always enjoyed web development, however it wasn't until April 2016 that I really started my adventure.
I mainly use open source curriculums like The Odin Project and my favourite, Free Code Camp.
I'm mainly focused on front-end development at the moment and have taught myself HTML - CSS - JavaScript - jQuery and Bootstrap to create simple, clean and responsive webpages. I'll admit I still got a lot to learn, but haven't we all?


When I'm not pushing code you can find me playing computer games, motsly world of warcraft(*sigh* I have a problem), finding and socialising in interesting pubs(I'm english what can I say?), drinking tea(again...english) catching up on my reading material(currently on the pragmatic programmer) and out walking my pupper.

tldr: self taught, html, css, javasript, jquery, ubuntu, computer gamer, dog walker, pub goer, book reader ...generally awesome.
